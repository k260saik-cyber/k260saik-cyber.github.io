Vision

1. 幅広い基礎知識の習得

プログラミング、インフラ、セキュリティといった各分野の基礎を固め、IT技術の全体像を把握できる、引き出しの多いエンジニアを目指します。資格取得も視野に入れ、体系的な知識を身につけていきます。

2. AIを活用した専門分野の確立

全ての分野に共通する「AIの活用」を自身の専門性とします。AIアシスタントをコーディングの相棒とするだけでなく、AI技術そのものを学び、開発や問題解決に応用できるスキルを磨いていきます。

3. 技術で、チームと社会に貢献する

身につけた技術は、自分だけのものではありません。チームの仲間と知識を共有し、助け合い、共に成長すること。そして、その技術を使って、人々の生活を少しでも豊かにするサービスやシステムを作ること。それが、私がエンジニアとして目指す最終的なゴールです。

RPG (IBM i)

私の原点です。リボーン。

SQL

データを制する者はシステムを制す。SQLは、あらゆるアプリケーションの裏側で活躍する、まさに「縁の下の力持ち」です。必要な情報を、いかに効率的に、正確に引き出すか。その奥深さに魅了されています。

HTML5

ウェブページの骨格を定義する、最も基本的な言語です。セマンティック（意味のある）なタグを意識することで、検索エンジンや支援技術にも優しい、質の高い構造を目指しています。

CSS3

FlexboxやGridといったモダンなレイアウト手法を使い、レスポンシブデザインに対応できるスタイリングを心がけています。`transition`や`transform`を使った、心地よいアニメーションの実装も好きです。

JavaScript

静的なページに命を吹き込む、魔法のような言語です。このサイトのスクロールアニメーションやモーダルウィンドウも、JSの力で動いています。ユーザーが触れて楽しい、インタラクティブな体験を作り出すことに関心があります。

Git & GitHub

このHPはgitでアップロードしました。

PHP

サーバーサイドの世界への入り口です。ユーザーの入力に応じて、動的にページを生成する仕組みは、Webアプリケーションの核心だと感じています。XAMPP環境で、データベースとの連携を学習中です。

Java

オブジェクト指向という、大規模開発における重要な考え方を学んでいます。「一度書けば、どこでも動く」という理念の通り、その汎用性の高さと、堅牢な設計思想に触れています。

C++

競技プログラミングに目覚め、水色目指して頑張ります。

CCNA

アプリケーションが快適に動くのは、安定したネットワークという土台があってこそ。CCNAの学習を通じて、データがどのように世界中を旅するのか、その壮大な仕組みを学んでいます。

Packet Tracer

高価なネットワーク機器に触れることなく、仮想環境で自由にネットワークを設計・検証できる素晴らしいツールです。頭の中の設計図を、実際に動かして試せるので、理解が格段に深まります。

TryHackMe

「守る」ためには、まず「攻める」方法を知らなければならない。TryHackMeでは、攻撃者の視点を体験することで、いかにして安全なシステムを構築すべきかを、実践的に学んでいます。

Progate

プログラミング学習の最初の「一歩」を踏み出させてくれた、恩人のようなサービスです。スライド形式で、自分のペースで直感的に学べるので、挫折することなく基礎を固めることができました。

YouTube

AI回しながら、youtubeを併用してITの勉強をしています。

AIアシスタント

Gemini、Copilot、ChatGPTをメインに1日12時間以上AIを活用しています。
